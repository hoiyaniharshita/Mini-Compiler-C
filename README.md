# Mini Compiler for Arithmetic Expressions

This project implements a basic compiler for arithmetic expressions using C programming.

The compiler performs lexical analysis, parsing, and intermediate code generation.

## Technologies Used

- C Programming
- Compiler Design Concepts

## Features

- Lexical analyzer using tokenization
- Deterministic finite automata for token recognition
- LL(1) grammar for arithmetic expressions
- Recursive descent parser
- Generation of three-address intermediate code
- Error detection using panic-mode recovery

## Workflow

1. Input arithmetic expression
2. Tokenization using lexical analyzer
3. Syntax parsing using recursive descent parser
4. Intermediate code generation

## Learning Outcomes

- Understanding compiler design
- Lexical analysis and parsing techniques
- Grammar implementation
